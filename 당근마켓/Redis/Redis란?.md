Redis는 데이터베이스, 캐시, 메시지 브로커 및 스트리밍 엔진으로 사용되는
오픈 소스(BSD 라이선스) 메모리 내 데이터 구조 저장소입니다.

Redis는 문자열, 해시, 목록, 집합, 범위 쿼리가 포함된 정렬된 집합,
비트맵, 하이퍼 로그 로그, 지리 공간 인덱스 및 스트림과 같은 데이터 구조를 제공합니다.

Redis에는 복제, Lua 스크립팅, LRU 제거, 트랜잭션 및 다양한 수준의 온디스크 지속성이 내장되어 있으며,
Redis Sentinel 및 Redis 클러스터를 통한 자동 파티셔닝을 통해 고가용성을 제공합니다. 

문자열에 추가하는 것과 같이 이러한 유형에 대해 원자성 작업을 실행할 수 있습니다.
해시 값을 증가시킵니다. 요소를 목록에 푸시합니다. 집합, 교집합, 합집합, 차이 계산; 또는 정렬된 집합에서 
가장 높은 순위를 갖는 구성원을 가져옵니다. 

Redis는 빠른 non-blocking 동기화와 net 분할 시 부분 재동기화를 통한 자동 재연결을 통해 비동기식 복제를 지원합니다. 

Redis에는 다음도 포함됩니다. 

트랜잭션
Pub/Sub
Lua Scripting
수명이 제한된 키
자동 장애 조치

대부분의 프로그래밍 언어에서 Redis를 사용할 수 있습니다. 

Redis는 ANSI C로 작성되었으며, Linux, *BSD, Mac OS X 등 대부분의 POSIX 시스템에서
외부 종속성 없이 작동합니다. 

Linux와 OS X는 Redis가 가장 많이 개발되고 테스트되는 두 가지 운영 체제이므로,
배포에는 Linux를 사용하는 것이 좋습니다. 

Redis는 SmartOS와 같은 Solaris 기반 시스템에서 작동할 수 있지만, 지원은 최선의 노력입니다.
Windows 빌드에 대한 공식적인 지원은 없습니다.
